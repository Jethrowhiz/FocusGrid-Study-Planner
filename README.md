A minimal, responsive study planner built with only HTML5 and CSS3.

## Overview
FocusGrid helps you plan focused study sessions with a calm, minimal UI.
It includes:

- Hero section with motivational headline and animated background shapes
- "Today's Focus" card (main goal, top 3 tasks, editable study time blocks)
- Weekly planner grid with selectable subjects, editable tasks, and priority tags
- Wellness check (sleep tracker, stress scale, break reminder)
- Printable Deep Work sheet (date, time, duration, tasks, reflections)

## How to use
1. Open `index.html` in your browser.
2. Edit fields directly on the page (no JavaScript required).
3. Use the print sheet for distraction-free sessions (click the print button).

## Developer notes
- Built with semantic HTML5 and modern CSS (Grid + Flexbox).
- Google Font: Inter.
- Print styles are included in `style.css` via `@media print`.
